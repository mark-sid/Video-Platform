# API for video sharing
A RESTful video sharing API built on **FastAPI** with support for asynchronous processing, caching, and authentication.
# Quick start
```
git clone https://github.com/mark-sid/Video-Platform.git
```
```
cd Video-Platform
```
```
docker  compose  up  --build
```
```
docker compose logs -f api
```
# Technologies used
- **FastAPI**
- Uvicorn
- **PostgreSQL**
- Asyncpg
- **SQLAlchemy**
- **Docker**, docker-compose
- Pydantic, pydantic-settings
- Alembic
- **Redis**
- SlowAPI
- FastAPI-Cache2
- Pytest, pytest-asyncio
